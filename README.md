# nixos-kotlin-dev-env

Slides for my talk of using nix to setup a portable and reusable development environment.

The slides are available at https://sengmann.github.io/nixos-kotlin-dev-env/.

The demo project is available at https://github.com/sengmann/dev-environments-playground.

## Starting the slides

1. clone repository
2. cd into folder
3. `direnv allow .`
4. `pnpm i`
5. `pnpm run start`
